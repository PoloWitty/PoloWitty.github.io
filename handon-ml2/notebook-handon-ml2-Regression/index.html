Regularized Linear Models

Ridge Regression

$$
J(\theta)= MSE(\theta) + \frac\alpha2\sum^n_{i=1}\theta^2_i
$$

$\alpha$ is not the learning rate but the hyperparameter of regularization

It’s really important to use feature scaling before fitting the model

the penalty sets the type of regularization term to use. (二阶范数)

or use Ridge class:

attention: the Ridge class use the closed-form solution of Normal Equation:
$$
\hat\theta=(X^TX+\alpha I)^{-1}X^Ty
$$
solver=’cholesky’ means that use the matrix factorization technique by Andre-Louis Cholesky.

Lasso Regression

$$
J(\theta)=MSE(\theta)+\alpha\sum^n_{i=1}|\theta_i|
$$

the difference between Ridge Regression and Lasso Regression

Lasso Regression tends to eliminate the weights of the least important features( namely set them to zero ). In other words, Lasso Regression automatically performs feature selection and outputs a sparse model( i.e. , with few nonzero feature weights )

The Lasso Regression is not differentiable at $\theta_i=0$ but Gradient Descent still work fine if you use a subgradient vector g instead where $\theta_i=0$. The gradient can be like this:
$$
\frac\part{\part\theta}J(\theta)=\nabla_\theta MSE(\theta)+\alpha *sign(\theta)\
\text{where } sign(\theta_i)=
\begin{cases}
-1 & \text{if } \theta_i<0\
0 & \text{if } \theta_i=0\
1& \text{if } \theta_i>0\
\end{cases}
$$

Logistic Regression

$$
p=\sigma(t)=\frac1{1+exp(-t)}
$$

the score t is often called the logit. Since $t=logit(p)=log(\frac p{1-p})$
$$
J(\theta)=-\frac1m\sum^m_{i=1}[y^{(i)}log(\hat p^{(i)})+(1-y^{(i)})log(1-\hat p^{(i)})]
$$

$$
\frac\part{\part\theta_j}J(\theta)=\frac1m\sum^m_{i=1}[\sigma(\theta^Tx^{(i)})-y^{(i)}]x^{(i)}_j
$$

Softmax Regression

$$
s_k(\mathbf x)=\mathbf x^T\theta^{(k)}
$$

$\theta^{(k)}$ is the parameter vector of class k. All these vectors are typically stored as rows in a parameter matrix $\Theta$
$$
\hat p_k=\sigma(s(\mathbf x))k=\frac{exp(s_k(\mathbf x))}{\sum^K{j=1}exp(s_j(\mathbf x))}
$$
K is the number of classes.
$$
\hat y=argmax_k \quad\sigma(s(\mathbf x))_k=argmax_k\quad s_k(\mathbf x)
$$

$$
J(\Theta)=-\frac1m\sum^m_{i=1}\sum^K_{k=1}y_k^{(i)}log(\hat p_k^{(i)})
$$

$y_k^{(i)}$ is the one-hot code of the class (0 or 1)

The gradient is :
$$
\nabla_{\theta^{(k)}}J(\Theta)=\frac1m\sum^m_{i=1}(\hat p_k^{(i)}-y_k^{(i)})\mathbf x^{(i)}
$$
Softmax Regression can be used to predict multiclass directly instead of using OvR method

When switch the LogisticRegression to Softmax Regression by setting the multi_class hyperparameter to multinomial, you must also specify a solver that support Softmax Regression, such as the “lbfgs” solver.
